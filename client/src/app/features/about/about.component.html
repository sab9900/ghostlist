What is Ghost List?

Ghost List is an end-to-end encrypted shared list app. You create lists, add items, chat with people you share with — and none of it is readable by anyone but you and the people you invite. Not even the server.

How it works

AES-256-GCM encryption

Every item, message and list name is encrypted on your device before it ever leaves. The server stores only ciphertext.

QR key exchange

Sharing uses ECDH P-256 key agreement. The encryption key travels only via QR code or link fragment — never through the server.

Real-time sync

Changes appear instantly on every connected device via SignalR — all still encrypted end-to-end.

Zero knowledge server

No accounts, no tracking, no ads. The server can't read your data. Your keys never leave your device.
