# Geist Sans fonts

Download from: https://github.com/vercel/geist-font/releases/latest
(Download the "Geist.zip" asset)

Place these files here:
- GeistVF.woff2          ← variable font (covers all weights, preferred)
- GeistMonoVF.woff2      ← mono variable (optional)

Or the individual weight files if you use the static package:
- Geist-Regular.woff2
- Geist-Medium.woff2
- Geist-SemiBold.woff2
- Geist-Bold.woff2
